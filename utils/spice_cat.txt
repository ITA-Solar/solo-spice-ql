Environment variables

  SPICE_DATA=../../.../

  SPICE_CAT_KEYWORDS=FILENAME,DATE-BEG,COMPRESS,OBS_ID,NWIN

  SPICE_CAT_KEYWORD_WIDTHS=FILENAME:x,DATE=BEG:y, ....

Invoke :

   Modal:     files = spice_cat()    - returns selection or !null
   Non-modal: spice_cat              - returns immediately

Filtering:

  Click green filter line OR column header to edit filter

  Text filters are regexps, range filters are min/max (inclusive)

Click/select/navigate with keys to a data cell: 
    Status message shows full content (also in cut buffer)

Selected files are highlighted in blue

Context menus (right-click):

  Right-click on header: 
     sort increasing/decreasing
     move column left/right

  Right-click on data cell:
     - Click filename... to do what? Start program w/file?
     - Click "filter on this value"

All files in a specific raster: right-click on filename, filter on, then clip filter
